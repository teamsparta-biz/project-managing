# 노션 페이지 공유 스킬

노션 페이지 URL과 이메일을 받아 사용자를 조회하고, 공유 단계를 안내하는 워크플로우입니다.

> **트리거**: "노션 공유해줘", "notion share", "노션 초대해줘" 등

---

## 필요한 정보 수집

스킬 시작 전 아래 정보가 있는지 확인하세요. 없으면 사용자에게 질문하세요.

- **노션 페이지 URL**: 공유할 페이지 주소 (예: `https://notion.so/workspace/Page-Title-abc123`)
- **추가할 사람 이메일**: 1명 이상 (여러 명일 경우 쉼표로 구분)
- **권한 수준**: 아래 중 선택 (기본값: 편집 가능)
  - `읽기 전용` — 내용만 볼 수 있음
  - `댓글 가능` — 읽기 + 댓글 작성
  - `편집 가능` — 내용 수정 가능
  - `전체 권한` — 공유 설정까지 변경 가능

---

## 실행 순서

### Step 1: 노션 페이지 정보 확인

`notion-fetch` 툴로 페이지 제목과 기본 정보를 가져옵니다.

- **입력**: 사용자가 제공한 페이지 URL
- **목적**: 페이지 제목 확인, 올바른 페이지인지 검증

페이지를 찾을 수 없는 경우 사용자에게 URL을 다시 확인하도록 안내합니다.

### Step 2: 이메일로 워크스페이스 사용자 조회

각 이메일에 대해 `notion-get-users` 툴로 사용자를 검색합니다.

- **입력**: `query` 파라미터에 이메일 주소
- **확인 항목**:
  - 워크스페이스 내 멤버인지 확인
  - 이름, 이메일, 사용자 ID 기록
  - 멤버가 아닌 경우 → 게스트 초대 방식임을 안내

### Step 3: 결과 정리 및 사용자 안내

아래 형식으로 대화창에 출력합니다:

```
📋 노션 공유 준비 완료

📄 페이지: {페이지 제목}
🔗 URL: {페이지 URL}

👥 추가할 사용자:
  • {이름} ({이메일}) — 워크스페이스 멤버 ✅
  • {이메일} — 워크스페이스 외부 (게스트 초대 필요) ⚠️

🔒 권한 수준: {선택한 권한}
```

### Step 4: 노션에서 공유 완료 안내

공유 방법을 단계별로 안내합니다:

```
📌 노션에서 공유를 완료하려면:

1. 아래 링크를 클릭해 페이지를 여세요:
   {페이지 URL}

2. 화면 오른쪽 상단 [공유] 버튼을 클릭하세요.

3. 이메일 입력창에 아래 이메일을 붙여넣으세요:
   {이메일 1}
   {이메일 2 (있는 경우)}

4. 권한 수준을 [{선택한 권한}]으로 설정한 뒤 [초대]를 클릭하세요.

완료되면 "{이름}님에게 {권한} 권한으로 초대 링크가 전송됩니다."
```

---

## 오류 대응

| 상황 | 대응 |
|------|------|
| 페이지 URL이 잘못됨 | URL 형식 확인 요청 (`notion.so/...` 형태인지 확인) |
| 사용자를 찾을 수 없음 | 워크스페이스 외부 사용자로 안내 — 게스트 초대 방식 설명 |
| 이메일이 여러 명인 경우 | 각 이메일을 순서대로 처리하고 결과 정리 |
| 페이지 접근 권한 없음 | 해당 페이지에 접근 권한이 있는지 사용자에게 확인 요청 |

---

## 완료 후 사용자에게 전달할 내용

- 조회된 사용자 목록 (워크스페이스 멤버 여부 포함)
- 페이지 직접 링크
- 노션 내 공유 버튼 위치 안내
- 설정할 권한 수준 재확인
